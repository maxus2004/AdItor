# VR-проект "Редактор параметров нейросетей"

Учебный проект по дисциплине «Разработка приложений на языке C#» (Unity)  
(МГТУ им. Н. Э. Баумана, кафедра «Компьютерные системы и сети»).  

Цель проекта — разработка VR-проекта для редактирования нейросетей. Редактор позволяет создавать схемы нейросетей из множества различных связанных блоков. Блоки являются слоями нейросети, можно выбирать их тип и настраивать параметры.

## Ключевые механики
- Добавление и удаление слоёв
- Редактирование параметров

### Редактируемые параметры:
- Размер слоёв
- Функция активации
- Другие параметры специфичные для отдельных типов слоёв

## Стек технологий
- Unity 6.3 LTS  
- XR Interaction Toolkit  
- OpenXR  
- C#  
- Git / GitHub
- Blender

## Структура проекта:
- Assets/*.cs - скрипты отвечающие за логику приложения
- Assets/Prefabs/ - префабы слоёв нейросетей и соединений
- Assets/Scenes/MainScene.scene - основная сцена проекта
- Assets/TextMesh Pro/ - файлы модуля TextMesh Pro, отвечающего за отрисовку текста интерфейс

## MVP
### На данный момент реализовано использование следющих типов слоёв:
- Input Layer
- Output Layer
- Dense Layer
- Recurrent Layer
- Convolution Layer
- Pooling Layer
- Dropout Layer
